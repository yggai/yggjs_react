## 命令简化
使用者安装方式：
```bash
npm install -g yggjs_react
```

创建基础模板：
```bash
yggjs_react create basic
```

使用别名创建基础模板：
```bash
ggr create basic
```

你需要自己执行pnpm build打包，然后执行本地化安装，使用安装后的命令验证以上流程。要确保能够正常创建项目，且项目能够正常启动。


## 代码规范
