## 🔍 快速查找

### 命令参考
```bash
# 创建项目
yggjs_react create my-app

# 查看版本
yggjs_react --version

# 获取帮助
yggjs_react --help
```

### 技术栈
- **Vite** - 构建工具
- **React 18** - UI框架  
- **TypeScript** - 类型系统
- **Zustand** - 状态管理
- **React Router** - 路由系统
- **pnpm** - 包管理器

### 项目结构
```
src/
├── components/     # 通用组件
├── pages/         # 页面组件
├── routers/       # 路由配置
├── store/         # 状态管理
├── App.tsx        # 主应用
└── main.tsx       # 入口文件
```
